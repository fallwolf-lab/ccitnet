# CCITNet 长春工程学院校园网自动登录

## 功能简介

图形化校园网自动登录工具，首次使用设置账号密码后，每次打开即可自动登录，无需手动操作。

## 使用说明

1. **首次启动** — 自动弹出设置窗口，输入校园网账号和密码后保存
2. **自动登录** — 保存成功后等待 4 秒倒计时，自动完成登录
3. **修改凭据** — 点击菜单栏「修改账号密码」即可更新

## 下载文件

文件已经打包成exe文件，直接下载zip文件解压即可运行
配置文件 `config.json` 会生成在 exe 同目录下，建议将 exe 放到一个空文件夹下。

## 自启动设置方法

- win + R 打开运行，输入 shell:startup
- 会打开一个文件夹，将想要启动的程序**快捷方式**放进文件夹
- 在设置里面搜索“启动”，可以看到开机启动项，确认已经打开
- 注意：此方法启动速度较慢

## 注意事项

- 账号密码以明文存储在本地 `config.json` 中，请注意保管
- 登录请求使用 RC4 加密传输密码
